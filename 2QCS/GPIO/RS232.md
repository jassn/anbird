## QUP Status
從 Device Tree 看該 UART node 的 status, 例如: 從 kalama target 看, `qupv3_se4_4uart status = OK`.


@kalama, qupv3_se4_4uart
```
cd /proc/device-tree/soc
```

Check status of device tree
```
cat qcom,qupv3_1_geni_se@ac0000/uart@a90000/status
```

Also, you can check */sys/class/tty* as well.
```
ls -al /sys/class/tty
```

## COM1 測試步驟
(in kalama) Check IO Expander Chip ID
```
gpioinfo | grep uart -B5
```

Set mode to RS232
```
gpioset -c 9 1=0 2=1
```
上面這個 adb shell 會停住.


另外再開一個 adb shell, 設定 baud rate...
```
stty -F /dev/ttyHS4 115200 raw
```

如果接收端已配置好, 就可以送出測試資料...
```
echo qwer123 > /dev/ttyHS4
```
